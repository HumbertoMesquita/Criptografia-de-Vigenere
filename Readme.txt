IMPLEMENTAÇÃO DA CIFRA DE VIGENÈRE

Este projeto implementa a cifra de Vigenère com Python.

EXECUÇÃO:
1. Execute o arquivo vigenere.py com Python
   python vigenere.py

2. Escolha uma opção (1 para criptografar, 2 para decriptografar).

3. Insira a mensagem e a chave desejada.

OBSERVAÇÕES:
- Apenas letras são cifradas. (Espaços e pontuações são preservados)
- A chave será automaticamente ajustada para o tamanho da mensagem.
